# ATC research index

Research documents describe hypotheses, evidence, and evaluation plans. They
do not change production behavior unless a separate architecture or decision
record explicitly accepts that change.

## Reading order

For the current research position, read the architecture and evaluation
program first, the Wave 1 horizon, both Wave 2 research challenges, and then
the integrated Wave 2 result. Wave 2 is a delta: where its later evidence
changes experiment order, the integrated result and its explicit decision
record guide the next preregistration without silently changing production
authority.
