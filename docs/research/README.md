# ATC research index

Research documents describe hypotheses, evidence, and evaluation plans. They
do not change production behavior unless a separate architecture or decision
record explicitly accepts that change.

## Current memory program

1. [ATC Memory Reliability Architecture](ATC_MEMORY_RELIABILITY_ARCHITECTURE.md)
   — umbrella two-plane product and research direction.
2. [ATC Memory Evaluation Program](ATC_MEMORY_EVALUATION_PROGRAM.md) —
   falsifiable benchmark program and CAOS endpoint.
3. [Wave 2 Memory Horizon Report](ATC_MEMORY_HORIZON_WAVE_2_2026-07-23.md) —
   fresh-horizon challenge current through July 23, 2026; prioritizes lossless
   programmatic logs, online/shift evaluation, poisoning admission, repair, and
   secure portability.
4. [Wave 1 Memory Horizon Report](ATC_MEMORY_HORIZON_REPORT_2026-07-23.md) —
   first fresh-horizon review; establishes the baseline ladder, harmful-memory
   gates, graph demotion, and derived-state closure priority.

## Reading order

For the current research position, read the architecture and evaluation program
first, then Wave 1, then Wave 2. Wave 2 is a delta report: where it changes
experiment order, its later evidence should guide the next Memory Lab
preregistration without silently rewriting accepted architecture or governance.
