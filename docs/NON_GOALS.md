# Explicit v1 non-goals

V1 excludes live location, heart rate and wearables, emergency or medical
behavior, family/organization accounts, multi-tenant SaaS, multi-master writes,
CRDTs, peer meshes, a general knowledge graph, arbitrary computer control,
autonomous permanent memory, credential ingestion, system-prompt extraction,
and hidden chain-of-thought extraction.

Embeddings, provider-specific extraction packages, a project-operated hosting
account, production-signed installers/notarization, and the Core-online
forwarding channel remain separate later milestones unless repository status
explicitly says they were implemented and exercised. V1 does include a
provider-neutral user-owned Edge blueprint, pairing flow, and OAuth MCP path;
the user still owns and pays for the external host.
